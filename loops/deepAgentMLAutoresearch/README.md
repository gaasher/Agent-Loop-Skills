# deepAgentMLAutoresearch

**Standard autoresearch, grounded in the literature via arXiv.**

[`standardMLAutoresearch`](../standardMLAutoresearch) with access to scientific literature.
Before (and between) experiments, the agent can search and read papers to ground its
proposals in prior work — pulling in known techniques and baselines instead of
reinventing them. Best for problems where the relevant methods are already published
and worth finding.
